# machado-assis-text-analysis
Text analysis of Machado de Assis’ novels using word frequency and Zipf’s Law to explore stylistic differences between Romanticism and Realism

# Machado de Assis Text Analysis 📚📊

This project explores the works of Machado de Assis through data analysis techniques, focusing on word frequency and Zipf’s Law.

By analyzing multiple novels from both his Romantic and Realist phases, the project identifies patterns in vocabulary usage and reveals a clear stylistic shift: from character-centered and emotional language to more abstract, psychological, and reflective terms.

## Key Insights
- Higher frequency of proper names and emotional terms in early (Romantic) works
- Increased use of abstract words like "life", "nothing", and "things" in Realist novels
- Evidence of greater lexical diversity in later works
- Zipf’s Law behavior observed across all texts

## Tools Used
- Python
- Pandas
- NLTK
- Matplotlib / Seaborn

## Goal
To combine literature and data science in order to better understand how language reflects deeper narrative and philosophical changes in literary works.
